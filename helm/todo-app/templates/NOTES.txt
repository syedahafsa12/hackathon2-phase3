Your cloud-native todo application has been deployed to Kubernetes!

ACCESSING THE APPLICATION:

1. Get the application URL:

   minikube service {{ .Release.Name }}-frontend -n {{ .Release.Namespace }}

   Or manually with Minikube IP:

   export MINIKUBE_IP=$(minikube ip)
   echo "Frontend URL: http://$MINIKUBE_IP:30000"

2. Open your browser to the URL above

VERIFYING THE DEPLOYMENT:

View all pods:
  kubectl get pods -n {{ .Release.Namespace }}

View all services:
  kubectl get services -n {{ .Release.Namespace }}

View backend logs:
  kubectl logs -n {{ .Release.Namespace }} -l app.kubernetes.io/component=backend -f

View frontend logs:
  kubectl logs -n {{ .Release.Namespace }} -l app.kubernetes.io/component=frontend -f

MANAGING THE APPLICATION:

Scale backend pods:
  kubectl scale deployment {{ .Release.Name }}-backend -n {{ .Release.Namespace }} --replicas=3

Update configuration:
  helm upgrade {{ .Release.Name }} ./helm/todo-app -n {{ .Release.Namespace }}

Uninstall:
  helm uninstall {{ .Release.Name }} -n {{ .Release.Namespace }}

AI-POWERED KUBERNETES MANAGEMENT:

Use k8sgpt for AI-assisted cluster analysis:

  k8sgpt analyze --namespace {{ .Release.Namespace }}
  k8sgpt analyze --explain --namespace {{ .Release.Namespace }}
